# THE POINT 一点 · 网站使用说明

**落地新西兰,一点全通** —— 高端华人一站式生活门户平台。
前台门户 + 用户/入驻方体系 + 审核流 + 后端 API + 数据库 + 管理后台,全部包含在这个文件夹里。

## 一、首次运行

1. 安装 Node.js **22.13 或更高版本**(https://nodejs.org 下载 LTS 版即可)
2. 在本文件夹打开终端,执行:

```
npm install
npm start
```

3. 打开浏览器访问:
   - 网站前台:http://localhost:3000
   - 管理后台:http://localhost:3000/admin/

首次启动自动建库并写入示例数据与 **9 大板块**(置业/顾问/教育/生活/集市/同城/招聘/文化/资讯,车辆类目并入集市)及初始分类,另附 6 家示例入驻方与 10 条示例内容便于演示(示例账号带「示例」字样,可在后台删除);老库升级会自动补齐新表新字段并自动把「车辆」板块并入「集市」,原有数据不受影响。

**默认管理员账号:`admin` 密码:`thepoint2026`** —— 登录后请立即在「设置」里修改密码。

## 二、平台机制

- **统一账号**:游客可注册(邮箱+密码),注册不分身份;普通用户可在「集市」发布二手闲置
- **入驻方**:用户通过「我要上一点」(页脚入口 / join.html)申请入驻;后台批准后获得自建主页(vendor.html),可自助发布服务、动态、活动、招聘
- **审核流**:一切公开内容(用户发布、入驻方发布、资料修改)都要在后台「内容审核 / 入驻审核」通过后才展示;编辑已发布内容会自动重新进入待审
- **一点认证 / 推荐位**:后台可为优质入驻方授予「✓ 一点认证」徽章与推荐位;内容可设「编辑精选」进入主页精选流
- **双语**:全站中/EN 切换(导航右侧按钮);内容为双语字段,EN 缺失自动回退中文
- **全站搜索**:导航搜索图标 → search.html,跨房源/入驻方/内容/资讯双语搜索
- **站内通知**:入驻与内容审核结果、收到留言时自动通知(导航铃铛红点 + 个人中心列表,打开即读)
- **联系方式防护**:入驻方与帖子的电话/微信/邮箱仅登录用户可见,游客显示「登录查看」引导注册
- **登录保持**:会话存数据库,重启服务不掉线;用户记住登录 30 天,管理员 12 小时

## 三、管理后台

- **概览**:用户数、待审入驻、待审内容等一屏统计
- **用户管理**:查看、封禁/解封、删除
- **入驻审核**:批准/驳回(附原因)/暂停,授予认证与推荐位,编辑资料
- **内容审核**:按状态筛选,通过/驳回/下架/设精选;「+ 官方发布」直接发文化文章、视频(支持 YouTube/Bilibili 链接)等
- **用户管控**:搜索/状态/注册时间筛选;详情(资料+入驻+全部发布);封禁(踢下线+内容主页自动下架,解封自动恢复)、重置密码、一键下架其内容、注销(级联删除,二次确认)
- **操作日志**:封禁/删除/审核等敏感操作全部留痕(谁/何时/对谁/做了什么),可检索
- **板块管理 / 分类管理**:板块(一级)与分类(二级)均存数据库,可随时增删改、排序、隐藏,前台自动生效
- **设置**:管理员密码、站点 URL(用于 sitemap/分享链接)、SMTP 邮件配置(填好后可接入 nodemailer 发邮件,接口已在 server.js `sendMail()` 预留;当前站内通知始终生效)
- 原有:房源、专家、资讯、伙伴、服务名录、客户留言(含入驻方留言)、均支持英文字段

## 四、文件结构


## 五、SEO 与性能

- 全部页面带双语 title / description / Open Graph(微信分享卡片同用 OG);详情页由 JS 动态更新分享标题图
- `/sitemap.xml` 由数据库实时生成(房源/内容/入驻方/文章全收录),`/robots.txt` 屏蔽 admin 与 api
- **上线后请在后台「设置」填写站点 URL**,sitemap 与分享链接才是正式域名
- 静态资源缓存 7 天、上传图 30 天,HTML 协商缓存;用户上传图自动前端压缩(≤1600px JPEG)
- 列表 API 已精简响应;图片懒加载;美化 404 页

## 六、常见问题

- **换电脑/部署**:整个文件夹拷走即可,`data.sqlite` + `uploads/` 就是全部数据
- **正式上线**:VPS 或 Railway/Render,`npm start`;建议加 Nginx + HTTPS,改默认密码,填站点 URL
- **数据量大了想换 PostgreSQL**:所有 SQL 集中在 server.js,迁移成本低
- **忘记管理员密码**:删除 `data.sqlite` 会重置全部数据,慎用;或联系开发者改库
- 根目录 `_t.txt`、`_sync_test.txt` 为无用遗留文件,可放心手动删除(程序删除被系统权限拦截)
